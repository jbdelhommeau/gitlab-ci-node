# gitlab-ci-node
Runner node for Gitlab Continuous Integration with lastest version of grunt, gulp, bower and yarn installed in global.

# Current Node version supported:
- 0.12.x
- 4.x.x
- 5.x.x
- 6.x.x
- 7.x.x
